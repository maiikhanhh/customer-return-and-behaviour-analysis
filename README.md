# Customer Behaviour and Return Analysis

## 📌 Business Problem
This project analyses customer purchasing and return behaviour in a UK-based e-commerce business to identify key drivers impacting revenue, profitability, and potential churn risk.

## 🛠 Tools
- R/Data Analysis
- DataLab (DataCamp)

## 📊 Key Insights
- Strong overall revenue growth is observed, especially during peak seasons, but is increasingly offset by rising return rates, limiting margin expansion
- Returns are highly concentrated in specific products, transactions, and markets, indicating localised operational issues rather than systemic problems
- Some international markets generate higher order value but bring significantly higher risk due to elevated return rates and bulk purchasing behaviour
- Bulk transactions introduce disproportionate financial and operational risk
- The Netherlands stands out as a high-value, low-risk market, ideal for scalable growth, while markets such as Switzerland exhibit high-return, high-volatility profiles
- Churn interpretation: Return behaviour could be used as a proxy for churn risk, with high-return markets indicating elevated potential customer disengagement and dissatisfaction, while stable markets demonstrate stronger retention patterns

## 🎯 Recommendations
- Adopt a market-differentiated strategy by scaling high-performing markets (like the Netherlands) while applying targeted controls in high-risk regions
- Treat return reduction as a core priority to improve both profitability and customer retention
- Strengthen product quality, fulfilment accuracy, and cross-border logistics to minimise returns
- Implement stricter monitoring and control mechanisms for bulk transactions to reduce financial exposure
- Optimise customer experience across delivery and post-purchase processes to improve retention and reduce churn risk
